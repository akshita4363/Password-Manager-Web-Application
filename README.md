# Password-Manager-Web-Application

This project is a Password Manager Web Application developed using Python and Flask. The application allows users to create an account, log in securely, and generate strong passwords based on customizable settings such as password length, strength, uppercase letters, and symbols.

Generated passwords are encrypted using the Fernet encryption algorithm before being stored in an SQLite database, ensuring that sensitive information remains secure. Users can also view their password history, delete stored passwords, and export their saved passwords to a CSV file for backup purposes.

The project demonstrates concepts such as web development using Flask, database management with SQLite, password encryption, and secure user authentication.
